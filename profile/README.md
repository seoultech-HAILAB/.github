# Coding Rule
작업, 유지보수의 편의성 증진과 효율적이고 강건한 애플리케이션 개발을 위해 `Coding Rule`를 제정한다.
## Naming
- Class = `PascalCase`
- variable, function, method = `camelCase`
- constant variable = `UPPER_CASE`
- filename, folder = `kebab-case`
